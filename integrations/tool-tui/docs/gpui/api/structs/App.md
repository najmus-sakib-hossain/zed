Expand description

Contains the state of the full application, and passed as a reference to a variety of callbacks.
Other [Context](struct.Context.html "struct gpui::Context") derefs to this type.
You need a reference to an `App` to access the state of a [Entity](struct.Entity.html "struct gpui::Entity").
