Expand description

Task is a primitive that allows work to happen in the background.

It implements [`Future`](https://doc.rust-lang.org/nightly/core/future/future/trait.Future.html "trait core::future::future::Future") so you can `.await` on it.

If you drop a task it will be cancelled immediately. Calling [`Task::detach`](struct.Task.html#method.detach "method gpui::Task::detach") allows
the task to continue running, but with no way to return a value.
