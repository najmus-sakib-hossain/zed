
# dx-www-packet

Zero-dependency binary protocol types for dx-www.

## Overview

This crate defines the core binary protocol types shared between server and client, with zero external dependencies for maximum portability.

## Installation

```toml
[dependencies]
dx-www-packet = { path = "../packet" }
```

## Features

- `std`
- Standard library support with serde and bincode
