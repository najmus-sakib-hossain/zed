
# dx-www-a11y

Compile-time accessibility auditor for the dx-www framework.

## Overview

This crate provides compile-time accessibility checking using OXC parser to analyze JSX/TSX components and ensure they meet accessibility standards.

## Installation

```toml
[dependencies]
dx-www-a11y = { path = "../a11y" }
```
