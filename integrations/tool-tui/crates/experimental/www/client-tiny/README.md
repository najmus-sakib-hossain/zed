
# dx-www-client-tiny

Ultra-tiny WASM runtime for dx-www - NO_STD Integer DOM (< 2 KB).

## Overview

This crate provides an extremely minimal WASM runtime with zero dependencies, targeting sub-2KB bundle sizes for performance-critical applications.

## Installation

```toml
[dependencies]
dx-www-client-tiny = { path = "../client-tiny" }
```
