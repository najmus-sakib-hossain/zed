
# Todo App Example

A complete todo application demonstrating forms and state management with dx-www.

## Features

- Add, complete, and delete todos
- Filter by status (all, active, completed)
- Persist to local storage
- Form validation

## Running

```bash
cd examples/todo-app dx dev ```
Open //localhost:3000 to see the app.


## Building


```bash
dx build ```
