
# dx-www-client

Sub-20KB WASM runtime for dx-www - pure FFI, no wasm-bindgen.

## Overview

This crate provides the main client-side WASM runtime for dx-www applications, optimized for minimal bundle size.

## Installation

```toml
[dependencies]
dx-www-client = { path = "../client" }
```

## Features

- `dev`
- Development mode with error handling and debugging
